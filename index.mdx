<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get up and running with Inkdown in minutes</p>
  
  <Steps>
    <Step title="Download Inkdown">
      Download the latest release for your platform from [GitHub Releases](https://github.com/inkdown/inkdown/releases).
      
      Available for macOS, Windows, and Linux.
    </Step>
    
    <Step title="Open a workspace">
      Launch Inkdown and select a folder to use as your workspace. All your notes will be stored locally in this folder.
      
      ```bash
      # Or start from the command line
      inkdown /path/to/your/notes
      ```
    </Step>
    
    <Step title="Start writing">
      Create your first note with `Cmd+N` (or `Ctrl+N` on Windows/Linux). Use the powerful CodeMirror 6 editor with live preview.
      
      <Accordion title="Editor Features">
        - Full markdown support with GFM
        - Vim mode available
        - Live preview panel
        - Syntax highlighting
        - Custom keyboard shortcuts
      </Accordion>
    </Step>
    
    <Step title="Extend with plugins">
      Install community plugins or build your own using the comprehensive Plugin API. Visit the [Plugin Development](/plugins/introduction) guide to learn more.
    </Step>
  </Steps>
</div>
