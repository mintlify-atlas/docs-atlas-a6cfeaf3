This guide will help you install Inkdown and create your first note in under 5 minutes.

## Prerequisites

<Note>
  Inkdown runs on **macOS**, **Windows**, and **Linux**. Make sure you're running a supported operating system.
</Note>

## Step 1: Install Inkdown

<Steps>
  <Step title="Download Inkdown">
    Visit the [GitHub Releases](https://github.com/inkdown/inkdown/releases) page and download the latest version for your platform:

    | Platform | File Type |
    |----------|----------|
    | macOS (Apple Silicon) | `.dmg` |
    | macOS (Intel) | `.dmg` |
    | Windows | `.msi` |
    | Linux | `.AppImage` |
  </Step>

  <Step title="Install the Application">
    **macOS:**
    1. Open the downloaded `.dmg` file
    2. Drag Inkdown to your Applications folder
    3. Right-click the app and select "Open" (first time only)

    **Windows:**
    1. Run the downloaded `.msi` installer
    2. Follow the installation wizard
    3. Launch Inkdown from the Start Menu

    **Linux:**
    1. Make the `.AppImage` executable:
    ```bash
    chmod +x Inkdown-*.AppImage
    ```
    2. Run the AppImage:
    ```bash
    ./Inkdown-*.AppImage
    ```
  </Step>

  <Step title="Launch Inkdown">
    Open Inkdown from your Applications folder, Start Menu, or command line.
  </Step>
</Steps>

## Step 2: Create a Workspace

<Steps>
  <Step title="Open or Create Workspace">
    On first launch, Inkdown will prompt you to open or create a workspace.

    A **workspace** is simply a folder on your computer where Inkdown will store your notes.

    <Tip>
      Choose a location you'll remember, like `~/Documents/Notes` or `~/Inkdown`.
    </Tip>
  </Step>

  <Step title="Select Workspace Folder">
    Click **"Create Workspace"** or **"Open Existing Workspace"** and select a folder.

    ```bash
    # Example workspace structure
    ~/Documents/Notes/
    ├── daily/
    ├── projects/
    └── archive/
    ```
  </Step>
</Steps>

## Step 3: Create Your First Note

<Steps>
  <Step title="Create a New Note">
    Use one of these methods:

    - Click the **"New Note"** button in the toolbar
    - Use keyboard shortcut: `Cmd/Ctrl + N`
    - Use Command Palette: `Cmd/Ctrl + P` → "Create new note"
  </Step>

  <Step title="Name Your Note">
    Give your note a name, for example:
    ```
    My First Note.md
    ```

    <Note>
      All notes are saved as `.md` (markdown) files. They're just plain text files that work with Git and any other text editor.
    </Note>
  </Step>

  <Step title="Start Writing">
    The editor will open with your new note. Start typing!

    ```markdown
    # My First Note

    This is my first note in Inkdown.

    ## Features I Love

    - Fast and responsive
    - Local-first
    - Markdown support

    ## Next Steps

    - [ ] Explore more features
    - [ ] Install plugins
    - [ ] Customize themes
    ```
  </Step>
</Steps>

## Step 4: Explore Basic Features

### Live Preview

Toggle live preview to see your markdown rendered in real-time:

- Click the **preview icon** in the toolbar
- Use keyboard shortcut: `Cmd/Ctrl + E`

### Command Palette

Access all commands quickly:

1. Press `Cmd/Ctrl + P`
2. Type to search for commands
3. Press Enter to execute

### File Navigation

Quickly find and open files:

1. Press `Cmd/Ctrl + O`
2. Type the file name
3. Press Enter to open

### Formatting

Use keyboard shortcuts for quick formatting:

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Bold | `Cmd + B` | `Ctrl + B` |
| Italic | `Cmd + I` | `Ctrl + I` |
| Heading | `Cmd + H` | `Ctrl + H` |
| Link | `Cmd + K` | `Ctrl + K` |
| Code Block | `Cmd + Shift + K` | `Ctrl + Shift + K` |

## Step 5: Customize Your Setup

<Tabs>
  <Tab title="Theme">
    Change the theme to match your preference:

    1. Open Settings: `Cmd/Ctrl + ,`
    2. Go to **Appearance**
    3. Choose **Dark**, **Light**, or install a custom theme

    <Tip>
      You can also install community themes from the Settings → Themes section.
    </Tip>
  </Tab>

  <Tab title="Editor">
    Customize editor behavior:

    1. Open Settings: `Cmd/Ctrl + ,`
    2. Go to **Editor**
    3. Configure:
       - Font size
       - Line height
       - Tab size
       - Vim mode
       - Auto-save
  </Tab>

  <Tab title="Plugins">
    Enable or install plugins:

    1. Open Settings: `Cmd/Ctrl + ,`
    2. Go to **Plugins**
    3. Enable built-in plugins:
       - Word Count
       - Quick Finder
       - Slash Commands
       - Live Preview
    4. Browse community plugins
  </Tab>
</Tabs>

## Quick Reference

### Essential Keyboard Shortcuts

<AccordionGroup>
  <Accordion title="File Management">
    - `Cmd/Ctrl + N` — New note
    - `Cmd/Ctrl + O` — Open file
    - `Cmd/Ctrl + S` — Save file
    - `Cmd/Ctrl + W` — Close tab
  </Accordion>

  <Accordion title="Editor">
    - `Cmd/Ctrl + E` — Toggle preview
    - `Cmd/Ctrl + B` — Bold
    - `Cmd/Ctrl + I` — Italic
    - `Cmd/Ctrl + K` — Insert link
  </Accordion>

  <Accordion title="Navigation">
    - `Cmd/Ctrl + P` — Command palette
    - `Cmd/Ctrl + \` — Toggle sidebar
    - `Cmd/Ctrl + Tab` — Switch tabs
  </Accordion>
</AccordionGroup>
